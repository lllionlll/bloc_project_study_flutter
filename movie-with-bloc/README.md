## Run Locally

Clone the project

```bash
  https://github.com/EnginBolat/movie-with-bloc.git
```

Go to the project directory

```bash
  cd movie-with-bloc
```

Install dependencies

```bash
  flutter pub get
```

Run App

```bash
  flutter run
```
